Contact management application made with :heart: and ionic

## Development setup

- Install [node](https://nodejs.org)

```shell
npm i -g ionic
git clone https://github.com/neko250/ionic-agenda
cd ionic-agenda/
npm i
ionic s -c
```

## Build and deploy

```shell
ionic build --prod
npm run deploy
```
